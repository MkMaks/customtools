Custom Tools is extension for pyRevit add-in for Autodesk Revit.

INSTALLING MANUAL
1. Copy "CustomTools.extension" folder to any location.
2. Run Revit instance and under pyRevit tab and open Settings in pyRevit panel.
3. In settings scroll down to "Custom Extension Directories" and add location of your copy of "CustomTools.extension" folder.
4. In pyRevit panel click Reload and wait after extension loads.
5. Set location of CSS file in pyRevit/Settings/Output Window Settings
